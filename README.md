# Job Portal 
This repository contains the backend API for the Zecser Job Portal, built using Django and Django Rest Framework (DRF). The system supports authentication, job posting, job applications, events, reactions (like/dislike), followâ€“unfollow system, and profile management.

---
## Features
* User registration & login (Simple JWT authentication)
* OTP based email verification
* Role based automatic profile creation for users 
* Profile based job posting and job apply
* Duplicate job applications are blocked
* Automatic job application confirmation email
* Event and image posting
* Like/Dislike feature for events with counts
* Comment posting feature for events
* User follow/unfollow features
* Search & Filter for jobs
* Postman API tests

## Tech stack
* Python
* Django
* Django Rest Framework
* JWT authentication
* React + vite
* SQLite
* Postman
---
## User Roles
* Jobseeker - Apply for jobs, view jobs,events,follow users
* Employer - Post jobs,manage posted jobs
* Company - Post jobs
 ---
## Authentication
* JWT authentication
* Login return access and refresh tokens
* Protected API requires #999999 Authorization: Bearer <access_token>
