# Job Portal 
---
A job portal system built in Python-Django as backend and DRF for API design,React and tailwindcss used for frontend design.

## Features
* User registration & login (JWT authentication)
* OTP based email verification
* Role based automatic profile creation for users (roles:- Company,Employer,Jobseeker)
* Profile based job posting and job apply
* Automatic job application confirmation email
* Search & Filter for jobs
* Postman API tests

## Tech stack
* Python
* Django
* JWT authentication
* React + vite
* Postman
